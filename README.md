# Trivia Quest 🎯

A mouse-driven trivia game built with Python and Pygame. Choose your category, 
set your difficulty, and test your knowledge across 20 topics — all through a 
clean, fully clickable interface.

## Features

- 20 trivia categories with color-coded buttons
- Multiple difficulty levels (Easy, Medium, Hard, and Mixed)
- Fully mouse-driven UI — no keyboard required
- Custom font and a polished royal blue and gold visual theme
- Instant feedback on every answer

## Tech Stack

- Python
- Pygame

## How to Run

1. Make sure Python and Pygame are installed
2. Clone this repository
3. Run the game:
```bash
python trivia_quest.py
```

## What I Learned

Building Trivia Quest gave me hands-on experience with Pygame's event system,
designing a state machine architecture, and implementing interactive UI elements
like clickable buttons using pygame.Rect and collidepoint().
